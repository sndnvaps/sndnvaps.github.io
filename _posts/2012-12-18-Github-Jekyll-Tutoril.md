### 安裝Jekyll
1. 安裝Ruby
2. 安裝devkit包
3. gem install jekyll
4. 弄一個網站雛型出來
5. cd your-site-path
5. jekyll --server  啟動Jekyll並且開啟本地伺服器
6. 用瀏覽器看 localhost:4000 就可以看編譯完的結果

### 安裝pygments
1. 安裝easy_install
2. easy_install pygments
3. 將C:\Python27\Script加入PATH  (Done)

### 啟動jekyll本地端
1. cd your_site
2. jekyll --server --auto
